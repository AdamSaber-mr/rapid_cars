
# RapidCars Website

This is the RapidCars website project.

## Development

Run `npm i` to install dependencies.

Run `npm run dev` to start the development server.

Run `npm run typecheck` to run TypeScript checks.

Run `npm run build` to create a production build.

Run `npm run preview` to preview the production build locally.

## Deploy to GitHub Pages

This repo is configured for the GitHub Pages project URL:

`https://adamsaber-mr.github.io/rapid_cars/`

Deployment command:

`npm run deploy`

Notes:

- The app uses hash routing for GitHub Pages compatibility (`/#/aanbod`, etc.).
- The deploy command publishes the `dist` folder to the `gh-pages` branch.
